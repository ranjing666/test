 # English + Coding Zero‑Base Camp

  This repository stores the curriculum, code examples and learning notes for a 
  **dual‑track beginner bootcamp** that teaches English and programming (Python)
   in parallel.

  ## 📅 Weekly Plan
  | Week | English Focus | Coding Focus |
  |------|---------------|--------------|
  | 1‑2  | Basic greetings, phonetics | What is a computer? Hello‑world script |
  | 3‑4  | Simple dialogues, self‑introduction | Variables, input / output |    
  | 5‑6  | Simple tenses (present & past) | Conditional statements, loops |     
  | …    | … | … |

  ## 📂 Repository Structure
  .
  ├── week01/
  │   ├── vocab.txt          # 150 basic words (EN + CN)
  │   └── hello.py           # First Python script
  ├── week02/
  │   ├── dialogue.md
  │   └── calculator.py
  └── README.md

  ## 🚀 How to Use
  1. **Read** the weekly notes (`weekXX/`).
  2. **Run** the Python examples: `python week01/hello.py`.
  3. **Commit** your own exercises and push them:
     ```bash
     git add .
     git commit -m "Week 3: added number‑guessing game"
     git push
