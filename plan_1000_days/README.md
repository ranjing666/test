# 1000 天计划目录

这个目录保存按天展开的 1000 天学习计划。

使用建议：
- 一次只看今天，不要一次性试图读完 1000 天。
- 每学完 1 天，就去 `study_logs/` 写一份记录。
- 每 10 天至少做一次 Git 提交，把学习痕迹保存到 GitHub。
- 真正学习细节时，去对应阶段的 `units/` 目录打开详细单元讲义。
