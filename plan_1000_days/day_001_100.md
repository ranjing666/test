# Day 001-100

阶段主题：基础入门

阅读方式：
- 只执行当前这一天。
- 每 10 天做一个小整合。
- 看不懂时，先跑例子，再回头读解释。

## 单元 001：认识电脑与开发环境（Day 001-010）
- 编程焦点：文件和文件夹；扩展名与路径；编辑器基本操作；运行 Python 文件
- 英语词汇：file, folder, open, save, run, code
- 英语句型：`I can open and run a Python file.`
- 本单元产出：写一份环境认识笔记并运行第一个脚本

### Day 001
- 编程：先理解 `文件和文件夹`，只求知道它是什么、为什么要学。
- 英语：朗读并认识这些词：file, folder, open, save, run, code。
- 任务：读 3 个最小例子；手敲 1 遍；再用中文写出对“认识电脑与开发环境”的第一理解。
- 产出：在学习日志中整理 5 个关键词，并说明 `文件和文件夹` 的作用。
- 检查：能用自己的话解释 `文件和文件夹` 是什么。

### Day 002
- 编程：围绕 `文件和文件夹` 做跟练，把昨天看到的内容变成自己能敲出来的代码。
- 英语：熟悉句型：`I can open and run a Python file.`，并尝试自己读 3 遍。
- 任务：重写昨天的例子，主动改 3 个小地方，例如文字、数字、顺序或变量名。
- 产出：得到一个能运行的练习版本，作为“写一份环境认识笔记并运行第一个脚本”的第 1 个小部件。
- 检查：能独立写出一个包含 `文件和文件夹` 的最小例子。

### Day 003
- 编程：开始学习 `扩展名与路径`，注意它和前一个知识点的关系。
- 英语：把今天遇到的高频词写成一个小词表，并给出中文解释。
- 任务：先看例子，再自己模仿 2 次，重点观察 `扩展名与路径` 在代码中的位置。
- 产出：补一段学习笔记，说明 `文件和文件夹` 和 `扩展名与路径` 分别负责什么。
- 检查：能指出一段代码里哪部分属于 `扩展名与路径`。

### Day 005
