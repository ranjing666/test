# 第 1 阶段：入门基础 学习日目录

这个目录按单元整理。
每个 `day_XXX.md` 都是一个完整学习日的入口。
- 阶段主页：`../README.md`
- 总导航：`../../学习顺序总导航.md`
- 配套英语：`../../编程英语同步学习/100天编程英语路线.md`

建议顺序：
1. 先打开当前 `day_XXX.md`。
2. 再进对应单元讲义 `02_units/unit_XXX.md`。
3. 再做工作簿 `03_workbooks/unit_XXX_workbook.md`。
4. 再完成模板代码 `04_code_templates/`。
5. 卡住时再对照 `05_code_solutions/`。
6. 再做 `06_quizzes/`。
7. 再写 `study_logs/dayXXX.md`。
8. 最后回到 `学习进度总看板.md` 打勾。
主线做完后，再补 `编程英语同步学习/100天编程英语路线.md` 对应的当天英语任务。
