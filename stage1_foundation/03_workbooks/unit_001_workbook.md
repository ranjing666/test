# 单元 001 工作簿：认识电脑与开发环境

- 对应范围：Day 001-010
- 配套单元讲义：`stage1_foundation/02_units/unit_001.md`
- 本工作簿目标：围绕“完成一个个人学习启动脚本”把这一单元真正练起来。

## 这份工作簿怎么用

建议顺序：
1. 先看单元讲义。
2. 再做这份工作簿里的练习。
3. 然后打开代码模板自己敲一遍。
4. 最后对照参考答案，但不要一开始就抄答案。

## 学完这一单元后你要会什么

- 知道什么是文件、文件夹、路径和 `.py` 文件。
- 会打开编辑器、保存文件、运行第一个 Python 脚本。
- 知道“改一处、运行一次、看一次结果”的最基本学习方法。

## 热身复习

1. 你现在能不能说出“文件”和“文件夹”的区别？
2. 你能不能找到一个 `.py` 文件并说出它是什么？
3. 你知不知道运行程序以后，屏幕上会发生什么？

## 跟着做

1. 打开编辑器，新建一个 `unit_001_practice.py` 文件。
2. 输入一行 `print("Hello")`，保存并运行它。
3. 把输出文字改成你自己的名字，再运行一遍。

## 基础练习

1. 写 3 行 `print()`，分别输出：你的名字、你的城市、你的学习目标。
2. 让程序再问你一个问题：`What is your name?`
3. 把程序输出改成中英各一行。

## 提升练习

1. 把你的程序做成一个“学习启动画面”，加上分隔线。
2. 尝试自己新建一个不同名字的 `.py` 文件并运行。

## 英语小练习

- 本单元词汇：`file`, `folder`, `open`, `save`, `run`, `code`
- 本单元句子：`I can open and run a Python file.`
- 请把句子读 3 遍，再用中文说出大意。
- 请任选 3 个词，自己造 3 个非常短的技术句子。

## 小项目作业

本单元最终小项目：`完成一个个人学习启动脚本`

2. 询问你的名字。
3. 询问你今天要学什么。
4. 输出一句鼓励自己的话。

## 参考答案方向

- 一开始不需要做复杂功能，只要能运行成功就是进步。
- 问用户问题时可以先记住 `input()` 这个名字，不要求立刻理解很深。
