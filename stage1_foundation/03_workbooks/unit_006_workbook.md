# 单元 006 工作簿：比较运算与布尔值

- 对应范围：Day 051-060
- 配套单元讲义：`stage1_foundation/02_units/unit_006.md`
- 本工作簿目标：围绕“完成一个年龄与分数检查器”把这一单元真正练起来。

## 这份工作簿怎么用

建议顺序：
1. 先看单元讲义。
2. 再做这份工作簿里的练习。
3. 然后打开代码模板自己敲一遍。
4. 最后对照参考答案，但不要一开始就抄答案。

## 学完这一单元后你要会什么

- 知道 True 和 False 是什么。
- 会做 `==`、`>`、`<` 等基本比较。
- 会写出“是否通过”“是否成年”这种判断逻辑。

## 热身复习

1. 什么时候程序需要回答“是或不是”？
2. 你能区分 `=` 和 `==` 吗？
3. 比较的结果为什么只有两种？

## 跟着做

1. 让程序判断 5 > 3 的结果。
2. 再让程序比较两个分数。
3. 最后输出是否通过。

## 基础练习

1. 判断年龄是否成年。
2. 判断分数是否及格。
3. 把判断结果输出成一句完整的话。

## 提升练习

1. 同时判断年龄和分数。
2. 输出多个比较结果。

## 英语小练习

- 本单元词汇：`true`, `false`, `equal`, `greater`, `less`, `check`
- 本单元句子：`I can check whether two values are the same.`
- 请把句子读 3 遍，再用中文说出大意。
- 请任选 3 个词，自己造 3 个非常短的技术句子。

## 小项目作业

本单元最终小项目：`完成一个年龄与分数检查器`

1. 输入年龄。
2. 输入分数。
3. 判断是否成年。
4. 判断是否及格。

## 自查清单

- [ ] 我会写 `==`。
- [ ] 我会做大小比较。
- [ ] 我知道比较结果会变成 True 或 False。
- [ ] 我会把判断结果输出成自然语言。

## 参考答案方向

- `=` 是赋值，`==` 是比较，这是特别容易混的点。
- 先直接打印比较结果，再慢慢把它写进更完整的句子里。
