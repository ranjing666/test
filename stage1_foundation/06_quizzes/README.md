# 第 1 阶段小测目录

这个目录保存第 1 阶段前 10 个单元的小测。建议学完一个单元后先独立回答，再对照讲义和代码。

## 文件列表

- `unit_001_quiz.py` - 认识电脑与开发环境
- `unit_002_quiz.py` - print 与注释
- `unit_003_quiz.py` - 变量与 input
- `unit_004_quiz.py` - 数字与四则运算
- `unit_005_quiz.py` - 字符串与格式化
- `unit_006_quiz.py` - 比较运算与布尔值
- `unit_007_quiz.py` - if / elif / else
- `unit_008_quiz.py` - while 循环
- `unit_009_quiz.py` - for 循环与 range
- `unit_010_quiz.py` - 函数入门
