# 单元 002 小测：print 与注释

## 问题

1. 请用自己的话解释：`会用 `print()` 输出多行内容。`
2. 请说出这个单元最终小项目：`完成一个带中英文注释的欢迎卡片程序` 的最小版本至少要包含什么。
3. 请把这句英文翻成中文：`This program prints text on the screen.`
4. 请从这些词里任选 3 个，自己写 3 句很短的话：print, text, string, comment
5. 请写一句：你在这个单元最容易出错的地方是什么？

## 参考答案方向

- 第 1 题不要求背标准定义，只要能说出它是干什么的就行。
- 第 2 题重点是先说出最小可运行版，不用一开始就追求复杂。
- 第 3 题大意对就行。
- 第 4 题句子很短也可以，只要体现词义。
- 第 5 题写真实情况，不要写空话。

## 通过标准

- 你能不用看资料回答至少 3 题。
- 你知道这个单元自己哪里还不稳。
- 你知道明天该继续补什么。
