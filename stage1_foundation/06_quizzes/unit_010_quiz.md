# 单元 010 小测：函数入门

## 问题

1. 请用自己的话解释：`知道函数是“把重复逻辑打包”。`
2. 请说出这个单元最终小项目：`完成一个由多个函数组成的小工具箱` 的最小版本至少要包含什么。
3. 请把这句英文翻成中文：`This function returns a useful result.`
4. 请从这些词里任选 3 个，自己写 3 句很短的话：function, call, parameter, return
5. 请写一句：你在这个单元最容易出错的地方是什么？

## 参考答案方向

- 第 1 题不要求背标准定义，只要能说出它是干什么的就行。
- 第 2 题重点是先说出最小可运行版，不用一开始就追求复杂。
- 第 3 题大意对就行。
- 第 4 题句子很短也可以，只要体现词义。
- 第 5 题写真实情况，不要写空话。

## 通过标准

- 你能不用看资料回答至少 3 题。
- 你知道这个单元自己哪里还不稳。
- 你知道明天该继续补什么。
