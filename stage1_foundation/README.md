# 第 1 阶段：入门基础

- 压缩 Day 范围：`Day 001-010`
- 对应旧计划：`旧 Day 001-100`
- 单元范围：`unit_001` 到 `unit_010`

## 这一阶段学什么

- 认识电脑、文件、编辑器和 Python 运行方式
- 学会 `print()`、变量、`input()`、数字、字符串
- 学会 `if`、循环、函数
- 做出最基础的小程序

## 这一阶段怎么学

固定按这个顺序：

1. `daily_guides/day_001.md`
2. `units/unit_001.md`
3. `workbooks/unit_001_workbook.md`
4. `code_templates/unit_001_template.py`
5. `code_solutions/unit_001_solution.py`
6. `quizzes/unit_001_quiz.md`
7. `study_logs/day001.md`

## 为什么改成 10 天合 1 天

- 打开目录时更干净
- `Day 001` 就直接对应 `unit_001`
- 旧版 10 天拆分细节还在单元讲义里，没有丢
