# 第 1 阶段：入门基础

这是正式学习路径的第一阶段。

目标不是“看懂很多”，而是“真的会做最基础的事”。

## 阶段目标

完成这一阶段后，你应该能做到：

- 知道什么是程序、代码、终端、文件
- 会运行 Python 程序
- 会使用 `print()`、变量、`input()`
- 能看懂最基础的报错
- 能写几个 10 到 30 行的小程序
- 能认识一批高频编程英语词汇

## 建议学习顺序

### 第 1 周
- 第 1 课：认识 Python、运行第一个程序
- 第 2 课：变量是什么
- 第 3 课：输入和输出
- 第 4 课：数字和字符串
- 第 5 课：小练习和复盘

### 第 2 周
- 条件判断 `if`
- 比较运算
- 小型问答程序

### 第 3 周
- 循环 `for` 和 `while`
- 重复任务的思想
- 小游戏练习

### 第 4 周
- 函数
- 简单项目整合
- 第一阶段复盘

## 详细学习顺序

现在这一阶段建议你固定按这个顺序学：

1. 先看 `units/unit_001.md` 这样的单元讲义
2. 再做 `workbooks/unit_001_workbook.md` 这样的工作簿
3. 然后自己完成 `code_templates/unit_001_template.py`
4. 最后再参考 `code_solutions/unit_001_solution.py`
5. 再做 `quizzes/unit_001_quiz.md`
6. 学完写 `study_logs/day001.md`

这样你不会陷入“只看不练”。

## 学习建议

### 学编程时
- 不要只看，必须亲手敲代码
- 每次改一个小地方，然后运行看变化
- 报错先读最后一行，再找我解释

### 学英语时
- 先认识最常出现的词
- 不要求一次全记住
- 在代码和注释里反复看、反复读

## 当前阶段的核心原则

- 慢一点没关系，乱学不行
- 先会输入和输出，再谈复杂逻辑
- 先理解例子，再自己改例子
- 先建立习惯，再追求速度
