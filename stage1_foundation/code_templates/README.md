# 第 1 阶段代码模板目录

这个目录保存第 1 阶段前 10 个单元的代码模板。建议先自己补、自己改，再运行。

## 文件列表

- `unit_001_template.py` - 认识电脑与开发环境
- `unit_002_template.py` - print 与注释
- `unit_003_template.py` - 变量与 input
- `unit_004_template.py` - 数字与四则运算
- `unit_005_template.py` - 字符串与格式化
- `unit_006_template.py` - 比较运算与布尔值
- `unit_007_template.py` - if / elif / else
- `unit_008_template.py` - while 循环
- `unit_009_template.py` - for 循环与 range
- `unit_010_template.py` - 函数入门
