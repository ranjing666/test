学完这一课，你要能做到：

- 知道什么是编程

- 会运行一个 Python 程序
- 看懂 `print()` 和 `input()` 的最基础用法

## 一、先不要怕，编程到底是什么

你可以把编程理解成一件很简单的事：

编程就是把一条一条指令写给电脑，让电脑按顺序做事。

比如：

- 你对电脑说：“显示一句话。”
- 你对电脑说：“问用户叫什么名字。”
- 你对电脑说：“把名字显示出来。”

这些“说话的方式”，就叫代码。

## 二、什么是 Python

Python 是一种编程语言。

你可以把它理解成：

- 中文：人和人交流
- 英文：人和人交流
- Python：人和电脑交流

为什么我们先学 Python？

- 写法相对简单
- 新手容易开始
- 资料很多
- 以后能做脚本、网站、数据分析、自动化

## 三、什么是 `.py` 文件

如果你看到这样的文件名：

```text
hello.py
```

它的意思就是：

- `hello` 是文件名字
- `.py` 表示这是一个 Python 程序文件

所以以后你看到 `.py`，就先想到一句话：

“这是 Python 代码文件。”

## 四、你的第一行重要代码

最经典的第一句代码是：

```python
print("Hello, World!")
```

它的意思非常简单：

“在屏幕上显示 `Hello, World!` 这句话。”

## 五、把这行代码拆开看

### 1. `print`

`print` 的意思是：

- 打印
- 显示
- 输出到屏幕

你现在先把它记成：

“让电脑把内容显示出来。”

### 2. `()`

括号表示：

`print` 要处理的内容写在括号里面。

### 3. `"Hello, World!"`

引号里的内容是文本。

在 Python 里，文本也常叫字符串，英文是 `string`。

你可以先把“字符串”简单理解成：

“被引号包起来的文字。”

## 六、你已经能做的第一件事

只要你会改引号里的文字，你就已经在写代码了。

例如：

```python
print("你好")
print("我在学习 Python")
print("My name is Tom")
```

这 3 行代码的本质完全一样：

- `print`
- 括号
- 引号中的文本

## 七、让程序问你名字

下面这段代码比 `print()` 多了一点点，但仍然很基础：

```python
name = input("What is your name? ")
print("Hello,", name)
```

### 第 1 行是什么意思

```python
name = input("What is your name? ")
```

拆开看：

- `input`：让程序等待用户输入
- `"What is your name? "`：程序显示给你看的提示文字
- `name`：一个变量，用来保存你输入的内容
- `=`：把右边得到的内容，交给左边的名字保存

你现在可以把“变量”理解成：

“一个带名字的小盒子，用来装数据。”

这里这个小盒子叫 `name`，里面装的是你输入的名字。

### 第 2 行是什么意思

```python
print("Hello,", name)
```

这行的意思是：

- 显示 `Hello,`
- 再显示变量 `name` 里的内容

如果你输入的是 `Li Hua`，最后会显示：

```text
Hello, Li Hua
```

## 九、最常见的初学者错误

### 错误 1：忘了引号

错误写法：

```python
print(Hello)
```

为什么错？

因为 Python 会以为 `Hello` 是一个名字，不是文字。

正确写法：

```python
print("Hello")
```

### 错误 2：用了中文括号

错误写法：

```python
print（"Hello"）
```

正确写法：

```python
print("Hello")
```

记住：

代码里尽量使用英文符号，不要用中文全角符号。

### 错误 3：引号没有成对出现

错误写法：

```python
print("Hello)
```

正确写法：

```python
print("Hello")
```
