# 第 1 课词汇表

这一课不追求背很多单词，只记最常见、最有用的。

## 本课核心词汇

| 英文 | 中文 | 你可以怎么理解 |
|------|------|----------------|
| code | 代码 | 写给电脑的指令 |
| program | 程序 | 可以运行的一组代码 |
| Python | Python 编程语言 | 我们现在学的语言 |
| file | 文件 | 保存在电脑里的内容 |
| run | 运行 | 让程序开始执行 |
| print | 输出、显示 | 把内容显示到屏幕 |
| input | 输入 | 让用户输入内容 |
| name | 名字 | 常用变量名 |
| text | 文本 | 文字内容 |
| string | 字符串 | 被引号包起来的文本 |
| line | 行 | 一行代码 |
| comment | 注释 | 解释代码的话 |
| learn | 学习 | 学东西 |
| practice | 练习 | 反复做 |
| error | 错误 | 程序出问题 |
| fix | 修复 | 把错误改好 |
| open | 打开 | 打开文件 |
| save | 保存 | 把修改存起来 |
| type | 输入、打字 | 在键盘上打出来 |
| understand | 理解 | 真正看懂 |

## 一定要会读的 5 个句子

### 1.
`I learn Python every day.`

中文：
我每天学习 Python。

### 2.
`Run the program.`

中文：
运行这个程序。

### 3.
`Print your name.`

中文：
输出你的名字。

### 4.
`What is your name?`

中文：
你叫什么名字？

### 5.
`I can fix the error.`

中文：
我可以修复这个错误。

## 本课最重要的英语目标

今天你只要能看懂下面这些，就已经很好了：

- `print`
- `input`
- `name`
- `run`
- `error`

## 记忆建议

- 不要一次硬背全部
- 先会看懂，再慢慢会拼写
- 学代码时顺便看到这些词，就算复习

你现在的目标不是“词汇量很大”，而是“看到高频词不陌生”。
