# 第 1 阶段工作簿目录

这个目录保存第 1 阶段前 10 个单元的工作簿。每份工作簿都包含练习、英语、小项目和自查清单。

## 文件列表

- `unit_001_workbook.md` - 认识电脑与开发环境
- `unit_002_workbook.md` - print 与注释
- `unit_003_workbook.md` - 变量与 input
- `unit_004_workbook.md` - 数字与四则运算
- `unit_005_workbook.md` - 字符串与格式化
- `unit_006_workbook.md` - 比较运算与布尔值
- `unit_007_workbook.md` - if / elif / else
- `unit_008_workbook.md` - while 循环
- `unit_009_workbook.md` - for 循环与 range
- `unit_010_workbook.md` - 函数入门
