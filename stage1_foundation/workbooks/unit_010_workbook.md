# 单元 010 工作簿：函数入门

- 对应范围：Day 091-100
- 配套单元讲义：`stage1_foundation/units/unit_010.md`
- 本工作簿目标：围绕“完成一个由多个函数组成的小工具箱”把这一单元真正练起来。

## 这份工作簿怎么用

建议顺序：
1. 先看单元讲义。
2. 再做这份工作簿里的练习。
3. 然后打开代码模板自己敲一遍。
4. 最后对照参考答案，但不要一开始就抄答案。

## 学完这一单元后你要会什么

- 知道函数是“把重复逻辑打包”。
- 会定义最基础的函数。
- 会传参数、会返回结果。

## 热身复习

1. 如果一段代码要重复写 5 次，怎样更省力？
2. 函数为什么比复制粘贴更好？
3. 参数和返回值分别在干什么？

## 跟着做

1. 写一个 `say_hello()` 函数。
2. 写一个 `add_numbers()` 函数。
3. 调用这两个函数看结果。

## 基础练习

1. 定义一个欢迎函数。
2. 定义一个求和函数。
3. 定义一个判断是否及格的函数。

## 提升练习

1. 把前面的小逻辑拆成多个函数。
2. 做一个“个人工具箱”程序。

## 英语小练习

- 本单元词汇：`function`, `call`, `parameter`, `return`, `reuse`, `useful`
- 本单元句子：`This function returns a useful result.`
- 请把句子读 3 遍，再用中文说出大意。
- 请任选 3 个词，自己造 3 个非常短的技术句子。

## 小项目作业

本单元最终小项目：`完成一个由多个函数组成的小工具箱`

1. 写欢迎函数。
2. 写加法函数。
3. 写判断函数。
4. 在主程序里调用它们。

## 自查清单

- [ ] 我知道函数存在的意义。
- [ ] 我会定义函数。
- [ ] 我会传参数。
- [ ] 我会用 `return` 返回结果。

## 参考答案方向

- 不要一开始就写很复杂的函数，先从 2 到 5 行的小函数开始。
- 把函数想成“有名字的小机器”，输入东西，输出结果。
