# 第 2 阶段项目包

这里放的是第 2 阶段里程碑单元的“可照着做”的项目包。

和 `04_code_templates/` 不同，这里更强调：

- 目录结构
- 最小可运行版本
- 运行步骤
- 下一步怎么扩展

## 当前项目包

### `unit_020_cli_learning_manager`

- 对应单元：`02_units/unit_020.md`
- 目标：完成一个命令行学习管理器
- 你会看到：数据存储、菜单、增删改查、完成状态

### `unit_030_min_flask_site`

- 对应单元：`02_units/unit_030.md`
- 目标：完成一个最小 Flask 网站
- 你会看到：Flask 应用入口、路由、模板、静态样式

## 建议怎么用

1. 先学对应单元讲义。
2. 再看这个项目包的 `README.md`。
3. 先跑通最小版本。
4. 再自己改名字、文案、样式或功能。

## 重要原则

- 先跑通，再改好。
- 先做最小版本，再加功能。
- 每改一步都运行一次，别堆着一起改。
