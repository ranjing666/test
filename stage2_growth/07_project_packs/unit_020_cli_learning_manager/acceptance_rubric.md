# Acceptance Rubric

## 达标版本标准

### Level 1: 跑通

- 程序能启动
- 菜单能显示
- 退出不会直接报错

### Level 2: 核心流程可用

- 可以添加任务
- 可以查看任务
- 可以标记完成
- 可以删除任务

### Level 3: 数据保存正确

- 退出后再次打开，任务仍然存在
- JSON 文件内容和界面显示一致
- 错误输入不会破坏数据结构

### Level 4: 能解释和改动

- 你能说明 `main.py` 和 `storage.py` 分别负责什么
- 你能自己增加一个字段，例如 `priority`
- 你能说清保存流程什么时候发生

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你开始真正掌握这个项目。
