# Manual Test Checklist

- [ ] 运行 `python main.py` 后能看到菜单。
- [ ] 输入 `1` 时，能看到当前任务列表。
- [ ] 输入 `2` 时，能成功添加一条任务。
- [ ] 输入 `3` 时，能把一条任务标成完成。
- [ ] 输入 `4` 时，能删除一条任务。
- [ ] 输入 `5` 时，能看到总数、完成数、未完成数。
- [ ] 退出程序后重新打开，之前保存的任务还在。
- [ ] 打开 `learning_tasks.json`，能看到数据结构和界面内容一致。

## 测试通过标准

- 菜单主流程能走通。
- 保存逻辑能生效。
- 错误输入不会直接让程序崩掉。
