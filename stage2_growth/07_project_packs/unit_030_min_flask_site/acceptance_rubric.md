# Acceptance Rubric

## 达标版本标准

### Level 1: 服务可启动

- `python app.py` 可以正常启动
- 浏览器能打开首页

### Level 2: 页面和接口可访问

- `/` 首页可访问
- `/about` 关于页可访问
- `/api/status` 返回 JSON

### Level 3: 模板与静态资源可用

- 模板能正确渲染变量或列表
- CSS 样式能生效
- 页面改动后刷新可见

### Level 4: 能自己扩展

- 你能再加一个新路由
- 你能改标题、文案、样式
- 你能解释模板和静态文件的区别

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你不只是会照抄，而是已经开始会改。
