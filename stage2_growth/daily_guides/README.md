# 第 2 阶段：成长进阶 学习日目录

这个目录按单元整理。
每个 `day_XXX.md` 都是一个完整学习日的入口。
- 阶段主页：`../README.md`
- 总导航：`../../学习顺序总导航.md`
- 配套英语：`../../编程英语同步学习/100天编程英语路线.md`

建议顺序：
1. 先打开当前 `day_XXX.md`。
2. 再进对应单元讲义 `units/unit_XXX.md`。
3. 再做工作簿、模板、答案和小测。
4. 最后写 `study_logs/dayXXX.md`。
