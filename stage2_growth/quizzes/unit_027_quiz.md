# 单元 027 小测：JavaScript 控制流

## 选择或简答题

1. 请用一句自己的话解释：`if 判断` 是干什么的？
2. `for 和 while` 为什么不是孤立存在的？它要和什么配合？
3. 如果你要做出 `完成一个简单网页计算器` 的最小版本，你会先保留哪两块能力？
4. 请把这句英文翻成中文：`I can control logic in JavaScript.`
5. 请从这些词里任选 3 个做短句：loop, array, object, function

## 参考答案方向

- 第 1 题：重点不是背标准定义，而是说出 `if 判断` 在当前单元里的作用。
- 第 2 题：要能说出 `for 和 while` 需要和前面的内容一起工作。
- 第 3 题：通常先保留 `if 判断` 和 `for 和 while`，再慢慢补完整。
- 第 4 题：大意说对就行，不要求字字死记。
- 第 5 题：句子很短也可以，只要能体现词义。

- 你能不用看资料说出至少 3 题。
- 你能把英文句子的大意说清楚。
- 你知道自己哪里还不稳，准备回去补哪一段。
