# Acceptance Rubric

## 达标版本标准

### Level 1: 基础环境可用

- 项目能启动
- 数据库能自动初始化
- 首页能正常打开

### Level 2: 用户流程可用

- 可以注册
- 可以登录
- 可以退出

### Level 3: 核心业务可用

- 登录后可以发文章
- 首页可以展示文章
- 数据重启后仍然存在

### Level 4: 结构和表达过关

- 你能解释用户表和文章表的作用
- 你能解释登录状态为什么存在 session 里
- 你能自己再加一个功能，例如删除文章或个人主页

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你已经开始具备后端项目理解力。
