# Acceptance Rubric

## 达标版本标准

### Level 1: 数据能跑通

- CSV 能被读取
- 脚本能正常执行
- 不会因为基本数据而报错

### Level 2: 结果能生成

- 终端会打印摘要
- `output/report.md` 会生成
- 报告里有核心指标

### Level 3: 结论有意义

- 你能指出一个最重要科目
- 你能指出一个学习高峰日
- 你能把结果用中文说清楚

### Level 4: 可迁移和可扩展

- 你能换成自己的数据
- 你能解释每个统计项怎么得来
- 你能继续加新的指标或图表

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你已经开始具备完整数据项目思维。
