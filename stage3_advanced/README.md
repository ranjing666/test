# 第 3 阶段：高级进阶

这一阶段对应 `Day 301-600`，重点是把你从“能写基础网页和小程序”推进到“会做后端、会读写数据库、会写接口、会测试、会部署、会做算法训练和数据项目”。

## 这一阶段现在学什么

这一阶段的实际单元范围是：

- `unit_031` 到 `unit_060`

整体路线是：

1. Flask 模板、表单、数据库、CRUD、登录、API
2. requests、pytest、部署、文档、博客项目
3. 问题拆解、复杂度、算法基础题型
4. Pandas、数据清洗、数据分析、可视化、抓取、ETL、数据项目

## 本阶段核心目标

完成这一阶段后，你应该能做到：

- 能写出更完整的 Flask 小网站
- 能理解 SQLite、CRUD、登录和 JSON API
- 会写最基础的测试和部署流程
- 会做一批典型算法与数据结构入门题
- 会使用 Pandas 做基础数据处理和分析
- 会做一个基础数据项目并整理结论

- 目标：形成数据处理和自动化入门能力

## 这一阶段的难点提醒

- 后端、算法、数据这三条线同时出现，容易觉得内容突然变多
- 其实重点不是同时学深，而是按单元一段一段推进
- 看到数据库、接口、部署这些词时，不要自己吓自己，先抓最小例子
- 算法题先求能做出版本，再慢慢优化

## 这一阶段的原则

- 先做通一个后端流程，再谈结构优化
- 先会一类算法题，再扩下一类
- 先做出一张图，再谈分析故事
- 每个单元都留下代码、笔记、测试或结果截图
