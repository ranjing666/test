# 第 3 阶段项目包

这里放的是第 3 阶段里程碑单元的项目包。

这一阶段开始，项目会明显更像真实开发任务。你不只是看概念，还要开始理解：

- 目录怎么组织
- 数据怎么存
- 页面和接口怎么配合
- 分析结果怎么整理成报告

## 当前项目包

### `unit_040_blog_system`

- 对应单元：`units/unit_040.md`
- 目标：完成一个可注册登录的博客系统
- 重点：Flask、SQLite、注册登录、发帖

### `unit_060_data_project`

- 对应单元：`units/unit_060.md`
- 目标：完成一个完整数据分析项目
- 重点：数据读取、汇总分析、生成报告

## 建议怎么学

1. 先把对应单元讲义看完。
2. 再读项目包里的 `README.md`。
3. 先跑通最小版本。
4. 最后再自己改字段、页面、报告结构。
