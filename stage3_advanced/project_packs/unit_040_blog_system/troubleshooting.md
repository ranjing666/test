# Troubleshooting

## 1. 提示没有 Flask 或 Werkzeug

处理方式：

```bash
pip install -r requirements.txt
```

## 2. 注册时报用户名重复

原因：

- 这个用户名已经在 `blog.db` 里存在

处理方式：

- 换一个用户名
- 或删除测试数据库后重新开始

## 3. 页面打不开或一直跳转

检查顺序：

1. 确认 Flask 服务还在运行
2. 确认地址是 `http://127.0.0.1:5000`
3. 看终端最后几行报错信息

## 4. 我想重置项目数据

处理方式：

- 关闭程序
- 删除 `blog.db`
- 重新运行 `python app.py`

## 5. 发布文章后首页没显示

检查点：

- 你是否真的登录了
- 标题和内容是否为空
- 终端里是否有数据库报错
