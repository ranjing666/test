# 单元 059 小测：ETL 管道入门

## 选择或简答题

1. 请用一句自己的话解释：`抽取数据` 是干什么的？
2. `转换数据` 为什么不是孤立存在的？它要和什么配合？
3. 如果你要做出 `完成一个小型 ETL 管道` 的最小版本，你会先保留哪两块能力？
4. 请把这句英文翻成中文：`An ETL pipeline moves and cleans data.`
5. 请从这些词里任选 3 个做短句：extract, transform, load, pipeline

## 参考答案方向

- 第 1 题：重点不是背标准定义，而是说出 `抽取数据` 在当前单元里的作用。
- 第 2 题：要能说出 `转换数据` 需要和前面的内容一起工作。
- 第 3 题：通常先保留 `抽取数据` 和 `转换数据`，再慢慢补完整。
- 第 4 题：大意说对就行，不要求字字死记。
- 第 5 题：句子很短也可以，只要能体现词义。

## 通过标准

- 你能不用看资料说出至少 3 题。
- 你能把英文句子的大意说清楚。
- 你知道自己哪里还不稳，准备回去补哪一段。
