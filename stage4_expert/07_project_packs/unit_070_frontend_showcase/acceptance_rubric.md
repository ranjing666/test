# Acceptance Rubric

## 达标版本标准

### Level 1: 页面能正常展示

- 页面能打开
- 主要区块能显示
- 样式不会完全错乱

### Level 2: 交互可用

- 筛选按钮能切换
- 项目卡片能正确变化
- 项目数量文案能跟着更新

### Level 3: 展示表达清楚

- 项目标题、描述、结果清晰
- 标签能帮助理解项目类型
- 页面在移动端仍可阅读

### Level 4: 能自己做展示升级

- 你能自己增加一张项目卡片
- 你能新增一个标签分类
- 你能解释数据数组和页面渲染的关系

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你已经开始具备前端作品包装能力。
