# 第 4 阶段项目包

这里放的是第 4 阶段里程碑单元项目包。

这一阶段的项目包会更强调：

- 前端展示效果
- 前后端接口对齐
- 工程目录结构
- 上线前的最小检查点

## 当前项目包

### `unit_070_frontend_showcase`

- 对应单元：`units/unit_070.md`
- 目标：完成一个可展示的前端作品
- 重点：页面结构、筛选交互、作品展示

### `unit_080_fullstack_demo`

- 对应单元：`units/unit_080.md`
- 目标：完成一个可访问的全栈项目
- 重点：FastAPI、CORS、前端调用接口、接口联调

## 建议怎么用

1. 先看对应单元讲义。
2. 再跑项目包里的最小版本。
3. 先理解主流程，再补样式和功能。
4. 最后自己做一次从零复现。
