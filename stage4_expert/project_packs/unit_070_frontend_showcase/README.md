# Unit 070 Project Pack: Frontend Showcase

这个项目包对应 [../../units/unit_070.md](../../units/unit_070.md) 的里程碑产出：`完成一个可展示的前端作品`。

## 你会做出什么

这是一个最小作品集页面，包含：

- 顶部介绍区
- 技能标签筛选
- 项目卡片列表
- 基础响应式布局

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：筛选和渲染逻辑
- `manual_test_checklist.md`：手动测试清单
- `troubleshooting.md`：常见问题排查

## 如何运行

直接双击 `index.html`，或者在这个目录开一个本地静态服务器后访问它。

如果你已经装了 Python，也可以执行：

```bash
python -m http.server 5500
```

然后浏览器打开：

```text
http://127.0.0.1:5500
```

## 你会练到什么

- 如何把项目数据和页面渲染分开
- 如何做按钮筛选
- 如何给作品加说明、标签和结果

## 下一步怎么升级
