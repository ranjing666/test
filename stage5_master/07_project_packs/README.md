# 第 5 阶段项目包

这里放的是第 5 阶段里程碑单元项目包。

这个阶段的重点不只是“再写代码”，而是把长期学习成果整理成可展示、可表达、可继续延伸的资产。

## 当前项目包

### `unit_090_specialization_kit`

- 对应单元：`02_units/unit_090.md`
- 目标：完成一个你主攻方向的专项项目
- 重点：方向选择、里程碑、交付清单、复盘

### `unit_100_reflection_kit`

- 对应单元：`02_units/unit_100.md`
- 目标：完成一份总复盘和下一阶段计划
- 重点：知识地图、教学表达、长期机制、未来路线

## 建议怎么用

1. 不要把这些模板当摆设。
2. 每写完一个模块，就填一部分。
3. 先求内容真实，再求语言漂亮。
4. 最后把它们整理成你自己的长期作品材料。
