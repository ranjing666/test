# Acceptance Rubric

## 达标版本标准

### Level 1: 方向清楚

- 你已经选定一个主方向
- 你知道这个方向要解决什么问题
- 你知道为什么现在先不做别的方向

### Level 2: 计划清楚

- 你写出了里程碑
- 你区分了 must have 和 nice to have
- 你知道每周要推进什么

### Level 3: 交付清楚

- 你知道要交什么材料
- 你知道 GitHub 仓库里要放什么
- 你知道怎样判断项目当前版本算完成

### Level 4: 表达清楚

- 你能用一段话介绍你的专项项目
- 你能解释为什么这样拆里程碑
- 你能说明下一阶段要补什么

## 当前单元建议目标

做到 `Level 3`，就算完成当前单元要求。  
做到 `Level 4`，说明你已经开始具备长期项目管理能力。
