# Deliverable Checklist

- [ ] I can explain the problem in one short paragraph.
- [ ] I can explain who the user is.
- [ ] I have a minimum working version.
- [ ] I can run the project from a clean start.
- [ ] I have a README with setup steps.
- [ ] I have screenshots, sample output, or demo notes.
- [ ] I know the current limitations.
- [ ] I know what version 2 would improve.
- [ ] I pushed the project to GitHub.
- [ ] I wrote a short reflection after delivery.
