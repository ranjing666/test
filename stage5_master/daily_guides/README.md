# 第 5 阶段：大师输出 压缩学习日目录

这个目录已经从旧版 10 天一拆的结构，改成了 1 个单元 = 1 个学习日。
每个 `day_XXX.md` 都是原来连续 10 天资料的合并入口。

建议顺序：
1. 先打开当前 `day_XXX.md`。
2. 再进对应单元讲义 `units/unit_XXX.md`。
3. 再做工作簿、模板、答案和小测。
4. 最后写 `study_logs/dayXXX.md`。
