# Unit 090 Project Pack: Specialization Kit

这个项目包对应 [../../units/unit_090.md](../../units/unit_090.md) 的里程碑产出：`完成一个你主攻方向的专项项目`。

## 这个包解决什么问题

很多人在后期最容易卡住的，不是不会写代码，而是：

- 不知道该选哪个方向
- 不知道怎么拆里程碑
- 不知道成果应该长什么样
- 不知道怎么判断项目是否真的完成

这个项目包就是帮你把这些问题拆开。

## 文件说明

- `direction_options.md`：方向选择参考
- `milestone_plan.md`：专项项目里程碑模板
- `deliverable_checklist.md`：交付物检查清单
- `progress_board.md`：执行过程中的周进度看板
- `acceptance_rubric.md`：当前单元完成标准
- `next_round_roadmap.md`：下一轮提升路线

## 推荐用法

1. 先在 `direction_options.md` 里选主攻方向。
2. 把选择结果填到 `milestone_plan.md`。
3. 每周对照 `progress_board.md` 更新状态。
4. 交付前用 `deliverable_checklist.md` 自查。

## 英语词汇

- `specialization`：专项方向
- `milestone`：里程碑
- `deliverable`：交付物
- `scope`：范围
- `review`：复盘
