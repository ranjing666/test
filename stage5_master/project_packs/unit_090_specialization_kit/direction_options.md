# Specialization Direction Options

## 1. Backend Direction

适合你如果更喜欢：

- 处理数据和业务逻辑
- 设计接口
- 研究数据库和稳定性

可以做的专项项目例子：

- API service for study tracking
- User system with auth and roles
- Small deployment dashboard

## 2. Frontend Direction

适合你如果更喜欢：

- 页面体验
- 交互流程
- 可视化表达

可以做的专项项目例子：

- Portfolio site with case studies
- Learning dashboard UI
- Admin panel prototype

## 3. Data Direction

适合你如果更喜欢：

- 清洗数据
- 分析规律
- 输出图表和报告

可以做的专项项目例子：

- Personal study analytics project
- Habit trend report
- ETL + dashboard demo

## 4. Automation Direction

适合你如果更喜欢：

- 提高效率
- 连接多个工具
- 用脚本减少重复劳动

可以做的专项项目例子：

- File organizer tool
- Report generator
- Batch data cleanup pipeline

## 选择规则

- 先选你愿意连续做 6 到 8 周的方向。
- 先选你最想展示给别人看的方向。
- 先选你最愿意继续深入的方向。
- 一次只选一个主方向，别贪多。
