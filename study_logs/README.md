# 学习记录说明

这个目录用来保存你的学习轨迹。

以后每学完一天，尽量保存一份记录。

## 命名建议

- `day001.md`
- `day002.md`
- `day003.md`

## 为什么一定要写学习记录

因为零基础最容易出现两个问题：

1. 学了以后很快忘
2. 不知道自己到底进步了什么

每天写几分钟记录，可以帮你做到：

- 看见自己真正完成了什么
- 复习当天最重要的内容
- 积累以后复盘要用的材料
- 在 GitHub 上留下连续学习痕迹

## 使用方式

- 新的一天开始时，复制 `daily_log_template.md`
- 改名为当天对应的 `dayNNN.md`
- 填写今天学了什么、哪里不会、明天做什么
- 每周结束时，填写 `weekly_review_template.md`
- 每月结束时，填写 `monthly_review_template.md`

如果你愿意，也可以让我直接帮你继续生成新的日志文件。
