  4. 评估与反馈机制

  1. 每两周一次小测
    - 英语：听写 20 词、口头对话 3‑5 min。
    - 编程：完成一道基础题目（如 “实现斐波那契数列”）并在 GitHub 上提交 PR。
  2. 每月一次项目评审
    - 用英文写项目概要、功能介绍、代码注释。
    - 由语言伙伴或同学进行代码审查（英文 review），提升技术与表达双能力。
  3. 自我复盘
    - 保存每周学习日志（英文），记录遇到的难点、解决办法。
    - 可在 Git 仓库的 README.md 中维护，形成长期可追溯的学习记录。

  开始行动

  1. 创建学习仓库（GitHub）
    - README.md 写上本训练营目标（英文）。
    - 每周创建分支 week01, week02 …，在对应分支里提交 单词列表、代码练习、项目代码。

  2. 下载核心教材
    - Python 官方文档（英文）: https://docs.python.org/3/
    - 《English Grammar in Use》 PDF（或购买纸质版）

  3. 安排首周日程（示例）
    - 周一/周三：英语 30 min（字母/发音）+ 编程 45 min（安装 Python、运行 print("Hello, world!")）
    - 周二/周四：英语 30 min（10 词卡）+ 编程 45 min（变量与输入）
