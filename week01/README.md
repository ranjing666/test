## 📖 第一课：Hello World

### 什么是编程？
编程就是用计算机能理解的语言，告诉它做什么事情。就像你用中文告诉朋友"帮我拿杯水"，编程就是用代码告诉计算机"在屏幕上显示一句话"。

**英文解释 (English Explanation):**
Programming is writing instructions for computers. Just like you tell your friend "get me water" in Chinese, you tell computers what to do using code.

### 你的第一个程序

打开 `hello.py` 文件，你会看到：

```python
# 我的第一个Python程序
# This is my first Python program

# 打印一句话到屏幕上
print("你好，世界！")
print("Hello, World!")
```

### 代码解释 (Code Explanation)

1. **注释 (Comments)**
   ```python
   # 这是注释
   ```
   - 以 `#` 开头的是注释
   - 计算机会忽略注释
   - 注释是给人看的，用来解释代码

2. **print() 函数**
   ```python
   print("你好，世界！")
   ```
   - `print` = 打印/显示
   - `()` = 括号里放要显示的内容
   - `""` = 引号里放文字（字符串）

### 运行程序 (Run the Program)

在终端输入：
```bash
python week01/hello.py
```

你会看到：
```
你好，世界！
Hello, World!
我叫什么名字？
我正在学习编程！
```
