## 📝 学习建议

### 记忆技巧
1. **分组记忆**：每天20个，分8天完成
2. **造句练习**：每个单词造一个句子
3. **代码应用**：在Python注释中使用新单词
4. **复习计划**：
   - 当天学完后复习1次
   - 第2天复习1次
   - 第4天复习1次
   - 第7天复习1次

### 发音练习
- 使用 [Google翻译](https://translate.google.com) 听发音
- 使用 [Forvo](https://forvo.com) 听真人发音
- 跟读3遍，录音对比

### 实战应用
在你的Python代码中使用这些单词：
```python
# Create a new variable (创建一个新变量)
name = "Tom"

# Print the output (打印输出)
print("Hello, my name is", name)

# This is easy to understand (这很容易理解)
```
