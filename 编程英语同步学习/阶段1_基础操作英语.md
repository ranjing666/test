# 第 1 阶段：基础操作英语

- 范围：`Day 001-010`
- 核心目标：先把最基础的文件、运行、输入、输出看懂，会读、会抄、会说最短句。

## 这一阶段英语到底练什么

- 先做到看见词不陌生。
- 再做到能读懂简单技术句子。
- 最后做到能写 1 到 3 句自己的说明。

## 这一阶段每天固定动作

1. 朗读当天句子 3 遍。
2. 抄写当天 6 个词。
3. 用 3 个词写 3 个短句。
4. 用中文解释一句话的技术意思。

## 这一阶段重点句子

- `I can open and run a Python file.`
- `This program prints text on the screen.`
- `My program can ask and answer simple questions.`
- `The result is shown on the screen.`
- `I can format a short message.`
- `I can check whether two values are the same.`
- `Please choose one option.`
- `Repeat the task until the answer is correct.`
- `For each number, I do one step.`
- `This function returns a useful result.`

## 这一阶段最常用的表达开头

- `I can ...`
- `This program ...`
- `The app ...`
- `Users can ...`
- `I can explain ...`

## 这一阶段高频词

- `file`, `folder`, `open`, `save`, `run`, `code`, `print`, `text`, `string`, `comment`, `line`, `title`, `variable`, `value`, `input`, `output`, `ask`, `answer`, `number`, `result`, `add`, `subtract`, `multiply`, `divide`, `word`, `sentence`, `message`, `format`, `length`, `space`

## 这一阶段收尾输出

- 能用 2 到 4 句英文说清自己今天写了什么。
- 能用 1 到 2 句英文写出程序作用。
- 能看懂当天句子的主要意思，不要求每个词都精确翻译。
