# 第 2 阶段：项目起步英语

- 范围：`Day 011-030`
- 核心目标：开始会描述列表、文件、Git、网页和表单，能用短句解释自己写的程序。

## 这一阶段英语到底练什么

- 先做到看见词不陌生。
- 再做到能读懂简单技术句子。
- 最后做到能写 1 到 3 句自己的说明。

## 这一阶段每天固定动作

1. 朗读当天句子 3 遍。
2. 抄写当天 6 个词。
3. 用 3 个词写 3 个短句。
4. 用中文解释一句话的技术意思。

## 这一阶段重点句子

- `I can keep many items in one list.`
- `I can sort and update my task list.`
- `A dictionary stores data with keys and values.`
- `I can clean and split text data.`
- `I can read from and write to a file.`
- `I can read an error and try to fix it.`
- `I can import code from another file.`
- `An object has data and behavior.`
- `Python has useful libraries for daily tasks.`
- `I can finish a small project step by step.`
- `I can save my work with Git.`
- `This HTML page has a clear structure.`
- `This form collects user information.`
- `I can style a page with CSS.`
- `My page can adapt to different screens.`
- `I can run simple JavaScript in the browser.`
- `I can control logic in JavaScript.`
- `I can update a page when the user clicks.`
- `I can build a small interactive frontend project.`
- `A route connects a URL to Python code.`

## 这一阶段最常用的表达开头

- `I can ...`
- `This program ...`
- `The app ...`
- `Users can ...`
- `I can explain ...`

## 这一阶段高频词

- `list`, `item`, `first`, `last`, `add`, `remove`, `order`, `sort`, `count`, `change`, `task`, `key`, `value`, `pair`, `data`, `detail`, `record`, `clean`, `split`, `join`, `find`, `replace`, `text`, `read`, `write`, `file`, `path`, `save`, `content`, `error`

## 这一阶段收尾输出

- 能用 2 到 4 句英文说清自己今天写了什么。
- 能用 1 到 2 句英文写出程序作用。
- 能看懂当天句子的主要意思，不要求每个词都精确翻译。
