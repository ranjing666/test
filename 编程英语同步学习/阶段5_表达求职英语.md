# 第 5 阶段：表达与求职英语

- 范围：`Day 086-100`
- 核心目标：重点转向文档、演示、作品集、面试和长期表达能力。

## 这一阶段英语到底练什么

- 先做到看见词不陌生。
- 再做到能读懂简单技术句子。
- 最后做到能写 1 到 3 句自己的说明。

## 这一阶段每天固定动作

1. 朗读当天句子 3 遍。
2. 抄写当天 6 个词。
3. 用 3 个词写 3 个短句。
4. 用中文解释一句话的技术意思。

## 这一阶段重点句子

- `A system design starts from requirements.`
- `Monitoring helps me find problems earlier.`
- `Security starts with careful input handling.`
- `I can contribute small improvements to open source.`
- `Focused practice helps me go deeper.`
- `My portfolio should show what I can build.`
- `Good writing makes technical work easier to learn.`
- `I can explain my project in English.`
- `My materials should show real work clearly.`
- `Practice makes interview answers clearer.`
- `A strong project starts with a clear plan.`
- `I can turn a plan into a working first version.`
- `I can improve the project based on testing.`
- `A finished project needs docs and polishing.`
- `Teaching is a strong way to learn deeply.`

## 这一阶段最常用的表达开头

- `I can ...`
- `This program ...`
- `The app ...`
- `Users can ...`
- `I can explain ...`

## 这一阶段高频词

- `system`, `scale`, `traffic`, `database`, `service`, `design`, `log`, `metric`, `monitor`, `alert`, `health`, `trace`, `secure`, `validate`, `secret`, `attack`, `auth`, `protect`, `open source`, `fork`, `pull request`, `review`, `issue`, `patch`, `specialize`, `focus`, `milestone`, `deliver`, `improve`, `portfolio`

## 这一阶段收尾输出

- 能用 2 到 4 句英文说清自己今天写了什么。
- 能用 1 到 2 句英文写出程序作用。
- 能看懂当天句子的主要意思，不要求每个词都精确翻译。
