# 高频编程英语词块

这不是全量词典，而是最值得先掌握的一批核心词。
先把这些词练熟，再去吃更大的英文材料。

- `open`：打开
- `save`：保存
- `run`：运行
- `read`：读取
- `write`：写入
- `check`：检查
- `fix`：修复
- `build`：构建
- `finish`：完成
- `repeat`：重复
- `choose`：选择
- `explain`：解释
